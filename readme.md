# GPS IOT 

Aplikasi web yang menunjukkan titik koordinat dari modul GPS uBlox NEO-6M-V2, ditampilkan ke dalam maps menggunakan API Here Maps.


# Teknologi

* PHP
* HTML, CSS, Bootstrap CSS framework
* Firebase Realtime Database
* Heroku
* Arduino Uno R3
* Modul SIM900
* API Here Maps
